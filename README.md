# 喝水提醒

一个轻量的 Windows 桌面喝水提醒工具，使用 Python 标准库实现，不需要安装额外依赖。

## 使用方式

1. 双击 `start_water_reminder.bat` 启动。
2. 窗口会置顶显示今日饮水进度、下次提醒倒计时和最近 7 天柱状统计。
3. 到点后会播放系统提示音，并弹出提醒。
4. 点击“喝 xxx ml”会记录一次饮水并重新计时。
5. 如果误记录，可以点“撤销 xxx ml”回退当天最近一次饮水记录，并恢复误点前的提醒倒计时。旧版本留下的无明细数据会按当前“每次饮水量”撤销。
6. 点击右上角“设置”，可以调整提醒间隔、每日目标和每次饮水量，保存后立即生效。
7. 点击右上角“收起”可以最小化；默认会收成悬浮胶囊。
8. 悬浮胶囊可以拖动，双击恢复主窗口，右键快速记录一次饮水。
9. 悬浮胶囊会显示倒计时和今日完成率；剩余 5 分钟内变成橙色，到点后红色闪动。
10. 设置里可以打开“开机自启动”，开启后会随 Windows 登录自动启动。
11. 每天第一次打开或跨过零点后，会自动开始记录新一天的数据。
12. “今日清零”只会清空当天记录，不影响其他日期的历史统计。
13. 关闭后再打开时，会按上次喝水时间继续倒计时；如果已经超过提醒间隔，会立即提醒。

## 数据保存

设置和每日饮水量会保存在 `water_reminder_config.json`：

- `daily_goal_ml`：每日饮水目标
- `drink_amount_ml`：每次饮水量
- `consumed_ml`：今天已经喝了多少
- `history`：按日期保存的每日饮水量
- `drink_log`：每次饮水记录，用于撤销误记录
- `last_drink_at`：上次喝水时间
- `next_reminder_at`：下次提醒时间
- `floating_ball_enabled`：是否最小化到悬浮球
- `autostart_enabled`：是否开机自启动
- `ball_x` / `ball_y`：悬浮球位置

## 资源占用

程序只有一个小窗口和每秒一次的倒计时更新，没有网络、数据库或浏览器进程，资源占用很低。

## 开机自启

如果需要开机自动启动，可以按 `Win + R`，输入 `shell:startup`，把 `start_water_reminder.bat` 的快捷方式放进去。
